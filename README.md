# ThinkBox

Private self-hosted learning platform for analytical thinking skills — guesstimation, problem structuring, quantitative reasoning, and related topics.

Built with Next.js 16 (App Router), React 19, Tailwind v4, and Zustand. Progress is stored in `localStorage`; there is no backend database.

## Development

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Deployment

Run it on your local by cloning the repo and running the dev server.
